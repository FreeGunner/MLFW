# MLFW

Machine Learning Framework implemented from scratch in Modern C++.

## Features

### Math Module

- Matrix
- CSV Loading
- Deep Copy
- Move Semantics
- RAII Memory Management
- Contiguous Memory Layout

- Linear Algebra
- Optimizers
- Loss Functions

- CUDA Backend
